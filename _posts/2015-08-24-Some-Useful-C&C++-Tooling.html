代码格式：

ClangFormat

ClangFormat

will help you codify your style and format your code for you to match the style. Simply write the code however you want,
and run the formatter on it before commiting it.

Linters:

Flint / Flint++

Flint

Flint++

Flint++是Flint的一个 fork， 其不包含任何第三方依赖库

While not quite full-on static analyzers, both can be used for finding potential issues in your code ahead of time.
Linters can look at individual files in isolation; you don’t have to wait for long recompiles like you would with a static analyzer.

静态分析：

Scan-build

Scan-build

Scan-view will emit and open an html file that shows a list of the errors it detected.
It will insert hyperlinks into the resulting document that step you through how certain conditions could lead to a null pointer dereference,
for example. You can also save and share those html files with others in the project.

运行时分析：

ASan and UBSan

其实包含Clang的一系列工具， See the list here

头文件分析：

Include What You Use

Include What You Use (IWYU)

It helps you find unused or unnecessary #include preprocessor directives.
It can also help cut down on recompiles by recommending forward declarations under certain conditions.

快速预编译：

ccache

ccache greatly improves recompile times by caching the results of parts of the compilation process.

distcc

distcc is a distributed build system.

Fuzzers:

American Fuzzy Lop

American Fuzzy Lop (AFL) is a neat program that performs fuzzing on programs
that take inputs from files and repeatedly runs the program, modifies the input trying to get full code coverage,
and tries to find crashes.

反编译:

gobjdump

If you really need to make sure the higher level code you’re writing is getting
translated into the assembly your expecting, gobjdump -S will intermix the
emitted binary’s disassembled assembly and the source code.
